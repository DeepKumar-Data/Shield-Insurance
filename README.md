# Shield Insurance Analytics

## Description
A comprehensive Power BI data visualization project focused on analyzing the performance metrics of Shield Insurance. This project processes multi-linked datasets spanning policy details, premium revenue, customer demographics, and claim settlements to provide a unified, interactive viewpoint of the business's operational health.

## Features
- **Customer Profiling:** Integrates customer age and city data to uncover demographic profiles driving policy sales.
- **Revenue Tracking:** Analyzes final premium amounts correlated with policy sales models (Offline-Agent, Offline-Direct, Online-App, Online-Website) to assess the most profitable channels.
- **Settlement Evaluation:** Tracks settlement percentages relative to policyholder age distributions, highlighting claim efficiency and potential risk avenues.
- **Benchmark Comparisons:** Monitors benchmarking features via integrated Excel checklists.
- **Mockup-to-Dashboard Translation:** Translates standard client design mockups into a fully interactive Power BI reporting environment.

## Tech Stack
- **Data Visualization & Modeling:** Power BI
- **Data Structuring / Referencing:** Microsoft Excel (`.xlsx`)
-- **Figma:** Mockups
